Cipher Encryption & Decryption Tool

A terminal-based classical cipher tool built to demonstrate foundational cryptography concepts in cybersecurity. Supports Caesar, Vigenere, ROT13, and Atbash ciphers with full encrypt and decrypt functionality. Features a built-in brute force engine that tries all 25 Caesar shifts, letter frequency analysis to help break unknown ciphers, and an educational reference manual explaining how each cipher works. Built with Python and Flask for the backend API, and vanilla HTML/CSS/JavaScript for the frontend. Styled with a retro amber phosphor terminal.
